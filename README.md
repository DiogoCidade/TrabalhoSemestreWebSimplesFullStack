# 📊 Sistema Web com PHP e MySQL

Projeto Full Stack desenvolvido utilizando PHP, HTML e MySQL, com integração entre frontend e backend e conexão com banco de dados relacional.

---

## 🚀 Funcionalidades

- Cadastro de registros
- Listagem de dados
- Edição de informações
- Exclusão de registros
- Conexão com banco via arquivo de configuração

O sistema implementa operações CRUD (Create, Read, Update e Delete), aplicando modelagem de dados e consultas SQL.

---

## 🛠 Tecnologias Utilizadas

- PHP
- HTML5
- CSS3
- MySQL
- SQL

---

## 🎯 Objetivo do Projeto

Aplicar na prática conceitos de desenvolvimento web com integração a banco de dados, simulando um sistema real de gerenciamento de informações.

---

## ⚙️ Como Executar

1. Instalar um servidor local (XAMPP, WAMP ou similar).
2. Importar o banco de dados no MySQL.
3. Configurar as credenciais no arquivo de conexão.
4. Colocar o projeto na pasta `htdocs`.
5. Acessar via navegador: `http://localhost/nome-da-pasta`.
